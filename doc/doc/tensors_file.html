Equivalent to create(Riak, Name, [], Attributes).

Create a tensors file handle with the given name and attributes. Will overwrite an existing file of the same name.

create/4

create(Riak::riak(), Name::binary(), Properties::proplist(), Attributes::dict()) -> {ok, File::tensors_file()}

Create a tensors file handle with the given name and attributes. Recognized properties: {block_size, int()} - The maximum size of an individual data chunk in bytes. Default is 1000000. {tree_order, int()} - The maximum number of children for an individual tree node. Default is 250.
