put_range/4

put_range(Riak::riak(), File::tensors_file(), Start::int(), Data::binary()) -> {ok, WrittenBlocks, NewFile}

Put range performs a synchronous write operation on the given tensors file. The write will start at the offset specified by Start and overwrite anything at that position with the contents of Data. Writes starting beyond the end of the file will occur at the end of the file. Tensors does not allow for gaps in a file.

truncate/3

truncate(Riak::riak(), File::tensors_file(), Start::int()) -> {ok, NewFile}

Truncates a file to the specified offset. All bytes beyond Start will no longer be readable in NewFile.
